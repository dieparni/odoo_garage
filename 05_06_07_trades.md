### Logique métier carrosserie

- Quand `requires_painting = True` et l'opération passe à `done`, créer automatiquement une `garage.paint.operation` liée si elle n'existe pas encore.
- Le schéma de dommages (vue SVG interactive avec zones cliquables) est un widget JS/Owl dans les vues — il met à jour les `damage_zone` visuellement. Prévoir le widget en phase 2 ; en phase 1 utiliser les selections standard.

```python
    unit_cost = fields.Monetary(string="Coût unitaire", currency_field='currency_id')
    total_cost = fields.Monetary(compute='_compute_cost', store=True, currency_field='currency_id')
    currency_id = fields.Many2one('res.currency', default=lambda self: self.env.company.currency_id)

    @api.depends('quantity', 'unit_cost')
    def _compute_cost(self):
        for rec in self:
            rec.total_cost = rec.quantity * rec.unit_cost
```

```python
class GarageMaintenancePlanItem(models.Model):
    _name = 'garage.maintenance.plan.item'
    _description = 'Point du plan d\'entretien'

    plan_id = fields.Many2one('garage.maintenance.plan', ondelete='cascade')
    name = fields.Char(string="Opération", required=True)
    interval_km = fields.Integer(string="Intervalle (km)", help="Ex: 15000")
    interval_months = fields.Integer(string="Intervalle (mois)", help="Ex: 12")
    last_done_km = fields.Integer(string="Dernier entretien (km)")
    last_done_date = fields.Date(string="Dernier entretien (date)")
    next_due_km = fields.Integer(
        string="Prochain (km)",
        compute='_compute_next',
        store=True,
    )
    next_due_date = fields.Date(
        string="Prochain (date)",
        compute='_compute_next',
        store=True,
    )
    is_overdue = fields.Boolean(
        string="En retard",
        compute='_compute_overdue',
    )

    @api.depends('last_done_km', 'interval_km', 'last_done_date', 'interval_months')
    def _compute_next(self):
        for rec in self:
            rec.next_due_km = (rec.last_done_km or 0) + (rec.interval_km or 0) if rec.interval_km else 0
            rec.next_due_date = rec.last_done_date + relativedelta(months=rec.interval_months) if rec.last_done_date and rec.interval_months else False
```

- Chaque module métier a ses opérations listées **en inline** dans le formulaire OR (onglets "Carrosserie", "Peinture", "Mécanique")

## Points d'attention

1. **Les zones de dommage** sont partagées entre carrosserie, peinture et les lignes de devis → factoriser en constante Python réutilisable (`DAMAGE_ZONES`)
2. **La cabine de peinture** est une ressource critique → son planning est géré dans le module planning (garage.workshop.post)
3. **Les véhicules électriques** nécessitent des habilitations spécifiques → vérifier `technician.has_ev_certification` avant affectation
4. **Les consommations peinture** doivent décrémenter le stock Odoo → lier `garage.paint.consumption` à `stock.move`
